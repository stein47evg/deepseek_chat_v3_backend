Извините за путаницу! Вот корректный код программы, помещённый в маркер:

## Как запустить

```bash
python triangle_area.py
```

**Пример работы (по трём сторонам):**
```
Расчёт площади треугольника

Выберите способ вычисления:
1. По основанию и высоте
2. По трём сторонам (формула Герона)
3. По двум сторонам и углу между ними

Ваш выбор (1/2/3): 2

Введите сторону a: 3
Введите сторону b: 4
Введите сторону c: 5

Площадь треугольника = 6.00 кв. ед.
```

## Итог

Файл `triangle_area.py` создан. Программа предлагает три способа вычисления площади треугольника, проверяет корректность введённых данных и выводит результат с точностью до двух знаков после запятой.
