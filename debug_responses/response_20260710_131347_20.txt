Вот программа для расчёта площади треугольника с выбором способа вычисления: по основанию и высоте, по трём сторонам (формула Герона) или по двум сторонам и углу между ними.

## Как запустить

```bash
python triangle_area.py
```

**Пример работы (по трём сторонам):**
```
Расчёт площади треугольника

Выберите способ вычисления:
1. По основанию и высоте
2. По трём сторонам (формула Герона)
3. По двум сторонам и углу между ними

Ваш выбор (1/2/3): 2

Введите сторону a: 3
Введите сторону b: 4
Введите сторону c: 5

Площадь треугольника = 6.00 кв. ед.
```

**Пример работы (по основанию и высоте):**
```
Ваш выбор (1/2/3): 1

Введите основание: 10
Введите высоту: 5

Площадь треугольника = 25.00 кв. ед.
```

**Пример работы (по двум сторонам и углу):**
```
Ваш выбор (1/2/3): 3

Введите сторону a: 5
Введите сторону b: 7
Введите угол между ними (в градусах): 30

Площадь треугольника = 8.75 кв. ед.
```

## Итог

Файл `triangle_area.py` создан. Программа предлагает три способа вычисления площади треугольника, проверяет корректность введённых данных (положительные числа, выполнение неравенства треугольника для формулы Герона) и выводит результат с точностью до двух знаков после запятой.
